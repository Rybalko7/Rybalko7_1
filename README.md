# PULSE STREAM | Портфолио фотографа и видеооператора

Персональное портфолио Александра Рыбалко — фотографа и видеооператора, специализирующегося на спортивной съемке.

**Дизайн:** Уникальный современный дизайн с футбольной эстетикой. Тёмная тема с неоновыми зелёными акцентами, анимациями и интерактивной галереей.

## 🚀 Технологии

- **Next.js 14** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **Framer Motion** - анимации и переходы

## 🎨 Особенности дизайна

- **Тёмная тема** — глубокий синий фон с зелёными акцентами
- **Градиенты** — фирменные градиенты от зелёного к синему
- **Анимации** — плавные появления, параллакс, hover-эффекты
- **Masonry галерея** — адаптивная сетка фотографий
- **Lightbox** — просмотр фото в полном размере
- **Фильтрация** — по категориям (матчи, процесс, эмоции)

## 🛠 Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

### 2. Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

### 3. Сборка для продакшена

```bash
npm run build
```

## 📤 Деплой на GitHub Pages

### 1. Создайте репозиторий на GitHub

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ВАШ_USERNAME/ВАШ_РЕПОЗИТОРИЙ.git
git push -u origin main
```

### 2. Установите gh-pages

```bash
npm install --save-dev gh-pages
```

### 3. Добавьте скрипты в package.json

```json
"scripts": {
  "predeploy": "npm run build",
  "deploy": "gh-pages -d out"
}
```

### 4. Задеплойте

```bash
npm run deploy
```

### 5. Настройте GitHub Pages

В настройках репозитория → Pages → Source → выберите ветку `gh-pages`
